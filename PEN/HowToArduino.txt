PEN で Arduino のプログラムを動かすための手順です。
この作業は初めて PEN で Arduino と通信させるために必要で、
1度この作業を行えば作業を行ったパソコンで再び設定する必要はありません。

[ Windows ]
(1)Arduino をパソコンに認識させるための作業（要管理者権限）

1.ArduinoIDE をダウンロード
　http://arduino.cc/en/Main/Software

2.ArduinoIDE を適当な場所に解凍する

3.Arduino を PC につなぐ

4.ドライバのインストールが始まるが
　ドライバが見つからないと言われるので、
　ArduinoIDE を解凍したディレクトリの drivers を参照し、
　ドライバをインストールする


(2)Java でシリアル通信できるようにするための作業（要管理者権限）

1.PEN の配布パッケージに「./lib」、「./lib64」があるか確認する

2.32bit の OS なら「./lib」を、64bit の OS なら「./lib64」を開く

3.「2」とは別に Java をインストールしたフォルダを開く
　例）C:\Program Files\Java\jdk6

4.Java をインストールしたフォルダから「jre」→「lib」→「ext」を開く

5.「ext」の中に「2」で開いたフォルダから「RXTXcomm.jar」をコピーする

6.Java をインストールしたフォルダから「jre」→「bin」を開く

7.「bin」の中に「2」で開いたフォルダから「rxtxSerial.dll」をコピーする


[ MacOS X ]
(1)Java でシリアル通信できるようにするための作業（要管理者権限）

1.PEN の配布パッケージに「./lib」、「./lib64」があるか確認する

2.32bit の OS なら「./lib」を、64bit の OS なら「./lib64」を開く

3.ファイルをを Java の標準クラスパスにコピーする
　32bit)
　% sudo cp ./lib/RXTXcomm.jar /Library/Java/Extensions/RXTXcomm.jar
　% sudo cp ./lib/librxtxSerial.jnilib /Library/Java/Extensions/librxtxSerial.jnilib

　64bit)
　% sudo cp ./lib64/RXTXcomm.jar /Library/Java/Extensions/RXTXcomm.jar
　% sudo cp ./lib64/librxtxSerial.jnilib /Library/Java/Extensions/librxtxSerial.jnilib


[ Linux ]



[共通設定]
Arduino と PEN が通信できるようにするための作業。
この作業は1度行えば Arduino にデータが書き込まれるため、
他の PC につないだ時に設定しなくてもよくなります。

1.Arduino を PC につなぐ

2.ArduinoIDE を起動する

3.ArduinoIDE の「ツール」→「マイコンボード」で
　接続している Arduino を選択

4.ArduinoIDE の「ツール」→「シリアルポート」で
　Arduino のシリアルポートを選択する

5.ArduinoIDE の「ファイル」→「スケッチの例」→
　「Firmata」→「StandardFirmata」を選択する

6.ArduinoIDE の「検証」のボタンを押し、スケッチをコンパイルする

7.ArduinoIDE の「マイコンボードに書き込む」を押し、スケッチを書き込む

8.ArduinoIDE を終了する
